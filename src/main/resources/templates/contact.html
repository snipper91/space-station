This is my first individual project since starting to learn web development in July 2017. As a whole this project reflects most of what I learned studying with LaunchCode and my continued studies afterwards. I'm very proud of what I've been able to accomplish in such a short time.

Some of the things showcased in this project include:

  • A back end programmed in Java utilizing the Spring Boot framework.
  • URL connections that exchange and parse JSON.
  • Bootstrap 4 used to arrange and style HTML elements.

Future updates will include:

  • A fully functional gallery of images taken from the ISS.
  • Real time tracking of the ISS on the home page.
  • Database manipulations handled from JavaScript so the page is not required to reload.
